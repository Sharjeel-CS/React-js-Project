🍽️ React Food Ordering App

A sleek and interactive food ordering app built with React.js, Redux, and Tailwind CSS, featuring a modern UI, category-based filtering, dynamic cart management, and a seamless user experience.

🚀 Features

✅ Browse Food Items – View a variety of delicious dishes with images, prices, and categories.✅ Category Filtering – Filter food by type (Veg/Non-Veg) and categories like Snacks, Beverages, etc.✅ Add to Cart – Easily add items to the cart with a single click.✅ Cart Management – Increase, decrease, or remove items from the cart dynamically.✅ Real-time Price Calculation – Automatic subtotal, tax, and delivery fee calculations.✅ Toast Notifications – Instant feedback when adding or removing items.✅ Smooth UI/UX – Responsive design with Tailwind CSS & animations.

🛠️ Tech Stack

Frontend: React.js, Redux Toolkit, Tailwind CSS, React Icons

State Management: Redux Toolkit

Notifications: React-Toastify

Deployment: Netlify/Vercel (Optional)
